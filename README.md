# SKORBORD - Canlı Skor ve Puan Durumu Takip Uygulaması

Skorbord, futbol dünyasındaki verileri (canlı skorlar, takımlar, fikstür, puan durumu) gerçek zamanlı ve modern bir arayüzle kullanıcılara sunmak için geliştirilen tam donanımlı bir ASP.NET Core projesidir.
